# Мультимодальные инварианты

## Onto8: 8-байтная структура

```
| 4 байта: объект | 4 байта: фон |
```

- **Объект** — выделенная сущность (фигура, имя, звук).
- **Фон** — контекст, в котором объект существует.
- Инвариант сохраняет структуру при трансформациях модальностей.

## Модальности

| Модальность | Реализация | Пример |
|------------|------------|--------|
| **Vision** | `vision.rs` | Цвет/форма + окружение |
| **Audio** | `audio.rs` | Тембр + реверберация |
| **Text** | `text.rs` | Имя + место действия |

## Синхронизация

- Модальности синхронизируются по **фазовой метке**.
- Рассогласование → искажение («всё-в-одном», проекция).

## Использование

```rust
let onto = Onto8::new(0x12345678, 0xABCDEF00);
let group = OnticGroup::new([onto.object; 4], onto.background);
```

> Инварианты — не данные, а **проекции физических законов в онтологическое пространство**.
