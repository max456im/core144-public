# Онтология: Lebesgue–Hammerstein Space

## Пространство сущностей

`LebesgueSpace` — основа всех фаз и групп:

```rust
struct LebesgueSpace {
    phase_point: u32,  // точка в фазовом пространстве
    intensity: u8,     // [0..255]
    coherence: f32,    // [-1.0..1.0]
}
```

## Свойства

- Порождает любые сущности с произвольными качествами и отношениями.
- Мера значимости: `measure = (intensity/255) * |coherence|`.
- Сохраняет структуру при трансформациях.

## Использование

- Каждая фаза — точка в `LebesgueSpace`.
- Каждая малоразмерная группа — когерентный поднабор точек.
